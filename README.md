## Boshlash

2. "Yangi Kolleksiya" yoki "New Deck" tugmasini bosing.
3. Kolleksiya nomini kiriting.
4. Kartalar qo‘shishni boshlang.

## Karta Qo‘shish

1. Kolleksiyani oching.
2. "Karta qo‘shish" tugmasini bosing.
3. Savol (Front) va Javob (Back) maydonlarini to‘ldiring.
4. "Saqlash" tugmasini bosing.

Misol:

Savol:
你好

Javob:
Salom

## O‘rganish Rejimi

1. "O‘rganish" bo‘limiga o‘ting.
2. Tizim kartalarni navbat bilan ko‘rsatadi.
3. Javobni eslashga harakat qiling.
4. "Javobni ko‘rsatish" tugmasini bosing.
5. Quyidagi variantlardan birini tanlang:

* Again — Bilmadim
* Hard — Qiyin bo‘ldi
* Good — Bilaman
* Easy — Juda oson

Tizim keyingi takrorlash vaqtini avtomatik hisoblaydi.

* Nomini o‘zgartirish

# 🇬🇧 English

## Getting Started

1. Open the application.
2. Click "New Deck".
3. Enter a deck name.
4. Start adding cards.

## Adding Cards

1. Open a deck.
2. Click "Add Card".
3. Enter Front and Back content.
4. Click "Save".

Example:

Front:
Hello

Back:
Salom

## Study Mode

1. Open the Study section.
2. Review cards one by one.
3. Try to recall the answer.
4. Click "Show Answer".
5. Select one of the following:

* Again
* Hard
* Good
* Easy

The system automatically schedules the next review.

## Managing Decks

Available actions:

* Rename
* Duplicate
* Delete
* Search cards
* Filter cards

## Backup & Restore

### Export

Import / Export → Export

A JSON backup file will be created.

### Import

Import / Export → Import

Select a previously exported JSON file.

All cards, settings, and statistics will be restored.

## Statistics

The Statistics page shows:

* Total cards studied
* Daily activity
* Accuracy rate
* Learning streak
* Review count

# 🇨🇳 中文

## 开始使用

1. 打开应用程序。
2. 点击“新建卡组”。
3. 输入卡组名称。
4. 开始添加卡片。

## 添加卡片

1. 打开卡组。
2. 点击“添加卡片”。
3. 输入正面和背面内容。
4. 点击“保存”。

例如：

正面：
你好

背面：
Salom

## 学习模式

1. 打开“学习”页面。
2. 按顺序学习卡片。
3. 尝试回忆答案。
4. 点击“显示答案”。
5. 选择以下选项之一：

* Again（不会）
* Hard（困难）
* Good（掌握）
* Easy（非常简单）

系统将自动安排下次复习时间。

## 卡组管理

支持以下操作：

* 重命名
* 复制
* 删除
* 搜索卡片
* 筛选卡片

## 数据备份与恢复

### 导出

导入/导出 → 导出

系统会生成 JSON 备份文件。

### 导入

导入/导出 → 导入

选择之前导出的 JSON 文件。

所有卡片、学习记录和统计数据都会恢复。

## 学习统计

统计页面显示：

* 已学习卡片数量
* 每日学习活动
* 正确率
* 连续学习天数
* 复习次数

## 离线模式

FlashLearn 支持：

* 离线学习
* 浏览器本地存储
* PWA 安装
* 自动备份

无需服务器即可使用。
